命令解析器

  • shell —— Unix操作系统
  • bash —— Linux操作系统

本质： 根据命令的名字，调用对应的可执行程序。

Linux Terminal 快捷键

  • 历史命令
    • 历史命令切换: （Ctrl+P/Ctrl+N）,（↑/↓）
  • 光标移动
    • 向前移动: Ctrl+B,←
    • 向后移动: Ctrl+F,→
    • 移动到最前方: Ctrl+A
    • 移动到最后方: Ctrl+E
    • 删除光标前面字符: Ctrl+H
    • 删除光标后面字符: Ctrl+D
  • 命令/路径补全: TAB

Linux 目录结构

  • 根目录
    1. root：管理员目录
    2. bin: Binary缩写，这个目录存放最常用的命令
    3. boot：存放启动Linux时使用的核心文件，包括一些连接文件以及镜像文件
    4. dev：Device缩写，该目录存放的是Linux外部的设备，Linux中访问设备的方式和访问文件方式是相同的
    5. etc：这个目录用来存放所有系统管理所需要的配置文件和子目录
    6. home：用户的主目录，Linuix中的用户都有一个自己的目录，一般以用户名命名
    7. lib：存放系统基本的动态连接共享库，其作用类似于Windows中的DLL文件，几乎所有应用都需要用到这些共享库
    8. lost+found：一般情况下为空，当系统非法关机后会存放一些文件
    9. media：Linux会自动识别一些设备，例如USB、光驱…当识别后，Linux会把识别的设备挂载到这个目录下
    10. mnt：手动挂载外设在此目录中。
    11. opt：主机额外安装的软件所摆放的目录，如安装一个Oracle数据库则可以放在此目录下。
    12. proc：虚拟目录
    13. sbin：存放管理员使用的系统管理程序
    14. usr: User software resource的缩写，用户的应用程序和文件都放在这个目录下，类似windows下的program files目录

用户目录

  1. 绝对路径: 从根目录开始写 /home/tommy/abc
  2. 相对路径: 相对于当前目录而言 bookslist.txt
    • .：当前目录
    • ..：上层目录
    • -：上次的目录
  3. tommy@ubuntu~
    • tommy: 当前登入的用户
    • @：在
    • ubuntu：主机名
    • ~：家目录
    • $：普通用户
    • #：超级用户
