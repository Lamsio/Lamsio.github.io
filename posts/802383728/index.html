    文件与目录

详细信息

我们可以通过$ ls -la查看当前目录下文件的信息。

常见命令

指令大全：Runnoob

  1. cd [路径]: 进入[路径]目录
  2. ls <参数> [路径]: 查看当前目录
  3. pwd：当前目录
  4. mkdir [文件路径] <参数>：在当前目录下创建文件夹/在当前目录下创建层级文件夹 $ mkdir aaa/bb/c -p
  5. rmdir [文件路径]：删除空的文件夹
  6. rm [文件路径] -r：以递归的方式删除文件夹及其底下的所有子内容
  7. touch [文件路径]：如果文件存在则更新文件日期，否则创建该文件
  8. cat [文件路径]：查看小型文件内容
  9. vi [文件路径]：最常用查看文件的指令，功能最全！
  10. cp [文件路径] [目的路径]：复制文件到指定路径中,若目的路径不存在则创建目的路径

文件与目录属性命令

wc [文件路径]

这条指令能够展示文本文件的行、词数、字节数。

od <参数> [文件路径]

这条指令能够展示二进制文本的内容

参数

  • c –> ASCII码
  • d –> 有符号十进制数
  • f –> 浮点数
  • o –> 八进制数
  • u –> 无符号十进制数
  • x –> 十六进制数

du -h [目录路径]

查看指定路径目录及其子目录大小

参数

  • h –> 符合人类阅读方式的格式

df -h [目录路径]

查看指定路径目录及其子目录的使用率。

which [命令名]

查看指定外部命令所处在的位置，像是cd命令是内建的，因此查不到。
